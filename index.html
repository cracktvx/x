Fire Stick Platform Installation

Step-by-step installation guide

English Español

1. Configure Developer Options

  • Go to Fire Stick Settings
  • Navigate to My Fire Stick
  • Select About
  • Click on the Fire Stick Name at least 7 times
  • Go back and choose Developer Options
  • Activate the 1st option and the 2nd option if available

2. Download the Downloader App

  • Hold the microphone button and say "DOWNLOADER"
  • Choose the 1st application (orange background with white down arrow)
  • Hit Download
  • After download, hit Open and Allow

3. Install the Platform

  • In the Downloader app, go to the Home tab
  • Enter this code:
96501
  • Hit Go
  • When downloaded, hit Next then Install
  • If needed: Go to Settings → Unknown Sources → Turn on Downloader → Go back twice → Hit Install again

4. Login

  • Open the installed app
  • Enter your membership username and password
